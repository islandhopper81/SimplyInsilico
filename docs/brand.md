# SimplyInsilico — Brand Guide

---

## Color System

### Primary Palette

| Name       | Hex     | Usage                                                        |
|------------|---------|--------------------------------------------------------------|
| Deep Teal  | #0F6E56 | Primary brand color. Center node, insilico wordmark, CTAs    |
| Mid Teal   | #1D9E75 | Inner orbit nodes, hover states, links                       |
| Light Teal | #5DCAA5 | Outer orbit nodes, accents, highlights                       |
| Pale Teal  | #9FE1CB | Backgrounds, tints, subtle fills                             |

### Neutral Palette

| Name       | Hex     | Usage                                           |
|------------|---------|-------------------------------------------------|
| Near Black | #1a1a18 | "simply" wordmark, body text                    |
| Mid Gray   | #888780 | Tagline, secondary text, rules                  |
| Light Gray | #B4B2A9 | Divider lines, borders                          |
| Off White  | #F1EFE8 | Page backgrounds, card fills                    |

---

## Typography

| Role       | Font                   | Weight  |
|------------|------------------------|---------|
| "simply"   | Georgia (serif)        | 400     |
| "insilico" | Courier New (monospace)| 700     |
| Tagline    | Georgia (serif)        | 400     |
| Body copy  | System sans-serif      | 400/500 |

The serif + monospace pairing is intentional:
- **Serif** = approachable, human, trustworthy
- **Monospace** = technical, precise, code-adjacent

## Dark Mode

For dark backgrounds, swap:

| Element       | Light Mode | Dark Mode |
|---------------|------------|-----------|
| "simply" text | #1a1a18    | #e8e6e0   |
| Tagline text  | #888780    | #6b6a64   |
| Rule line     | #B4B2A9    | #444441   |

Brand teal colors remain unchanged in dark mode — they hold contrast on both light and dark backgrounds.

---

## Single Color Version

For contexts requiring a single flat color (embroidery, stamps, watermarks):

- Use **#0F6E56** (Deep Teal) on white backgrounds
- Use **#9FE1CB** (Pale Teal) on dark backgrounds
- All opacity layers collapse to solid fills at 100%
